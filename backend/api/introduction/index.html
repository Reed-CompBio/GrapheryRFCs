    Introduction

    This folder is intended to specify the API backend server exposes to the public. It currently has two sections, graphql and ws. In the graphql folder, three files, types.md, queries.md, and mutations.md details the a set of API following the GrahQL specs. In the ws folder, the only file denotes a set of API relying on pure websocket.
